# 🎬 Movie Recommender NLP

An intelligent movie recommendation system that suggests relevant movies to
users based on content similarity and natural language processing techniques.


## ✨ Features

- 🎯 Content based movie recommendations
- 🔍 NLP preprocessing — tokenization and stemming
- 📐 Cosine similarity for movie matching
- 🎬 Top 10 similar movies instantly
- ⚡ Real-time results


## 🧠 Tech Stack

- Python
- scikit-learn
- NLTK
- Pandas
- NumPy
- Streamlit


## 📊 Models Used

- TF-IDF Vectorizer
- Cosine Similarity
- Count Vectorizer
- Bag of Words


## ▶️ Run Locally

git clone https://github.com/dheeraj815/Movie-RecommenderNLP.git
cd Movie-RecommenderNLP
pip install -r requirements.txt
streamlit run app.py
